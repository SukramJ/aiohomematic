*[CCU]: Central Control Unit - The Homematic central unit (CCU2, CCU3, or RaspberryMatic)
*[HmIP]: Homematic IP - The newer protocol for Homematic devices
*[HM]: Homematic - The classic BidCos-RF protocol
*[RF]: Radio Frequency - Wireless communication protocol
*[XML-RPC]: XML Remote Procedure Call - Protocol used by CCU for communication
*[JSON-RPC]: JSON Remote Procedure Call - Protocol used by CCU-Jack and CUxD
*[HA]: Home Assistant - Open-source home automation platform
*[API]: Application Programming Interface
*[DI]: Dependency Injection - Design pattern for loose coupling
*[ADR]: Architecture Decision Record - Documentation of design decisions
*[RSSI]: Received Signal Strength Indicator - Measure of wireless signal strength
*[eTRV]: Electronic Thermostatic Radiator Valve
*[I/O]: Input/Output
*[UI]: User Interface
*[MQTT]: Message Queuing Telemetry Transport - Lightweight messaging protocol
*[CUxD]: CUx-Daemon - Universal interface daemon for CCU
*[async]: Asynchronous - Non-blocking programming pattern
*[asyncio]: Python's asynchronous I/O framework
