_[CCU]: Central Control Unit - The Homematic central unit (CCU2, CCU3, or RaspberryMatic)
_[HmIP]: Homematic IP - The newer protocol for Homematic devices
_[HM]: Homematic - The classic BidCos-RF protocol
_[RF]: Radio Frequency - Wireless communication protocol
_[XML-RPC]: XML Remote Procedure Call - Protocol used by CCU for communication
_[JSON-RPC]: JSON Remote Procedure Call - Protocol used by CCU-Jack and CUxD
_[HA]: Home Assistant - Open-source home automation platform
_[API]: Application Programming Interface
_[DI]: Dependency Injection - Design pattern for loose coupling
_[ADR]: Architecture Decision Record - Documentation of design decisions
_[RSSI]: Received Signal Strength Indicator - Measure of wireless signal strength
_[eTRV]: Electronic Thermostatic Radiator Valve
_[I/O]: Input/Output
_[UI]: User Interface
_[MQTT]: Message Queuing Telemetry Transport - Lightweight messaging protocol
_[CUxD]: CUx-Daemon - Universal interface daemon for CCU
_[async]: Asynchronous - Non-blocking programming pattern
_[asyncio]: Python's asynchronous I/O framework
